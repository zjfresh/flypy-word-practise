# 小鹤音形Mac练习

> 基于 electron-vue 开发

![](./static/product.png)

> 输入框中修改完成后，皆可回车刷新或提交

#### Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:9080
yarn run dev

# build electron application for production
yarn run build


# lint all JS/Vue component files in `src/`
yarn run lint

```
